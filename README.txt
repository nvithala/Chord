
Steps to execute without SBT:
1.  Unzip the folder, go to the location where project3.scala is placed

2.  On command prompt type to compile the program: scalac project3.scala 

3.  To execute : scala project3 <numNodes> <numRequests>


Steps to execute with SBT:
1.  Unzip the folder, go to the location where project3.scala is placed

2. On command prompt type to compile the program: sbt run "<numNodes> <numRequests>"


Libraries used:
akka-actor_2.11 version:2.3.13
config-1.2.1 

Simulated Chord Protocol. Implemented node joins and finger table formation and updation corresponding to the nodes.
